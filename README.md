# Airbnb Data Analysis & Dashboard 🏠📊

## 📌 Project Overview

This project presents an end-to-end analysis of Airbnb listings using **Python for exploratory data analysis (EDA)** and **Power BI for interactive dashboarding**.

The analysis focuses on uncovering pricing trends, neighborhood dynamics, host growth patterns, and overall Airbnb market performance.

---

## 🎯 Objectives

* Analyze Airbnb pricing across neighborhoods
* Understand the relationship between accommodation size and price
* Study host growth trends over time
* Identify market patterns and pricing behavior
* Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies

* **Python (Pandas, Matplotlib)** – Data cleaning & analysis
* **Power BI** – Dashboard creation & visualization

## 📈 Key Insights

* Airbnb listings experienced rapid growth between 2012–2016
* Average prices increased over time in premium areas
* Entire homes dominate the Airbnb market
* Larger accommodations generally have higher pricing
* Pricing data contains significant outliers and skewness
